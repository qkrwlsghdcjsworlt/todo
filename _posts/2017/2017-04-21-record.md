~~우선
`sudo apt-get install xdotool`
설치후~~
설정에 keyboard 에서\\
name : end\\
command : `bash -c "xdotool getactivewindow key End"`\\
name : home\\
command : `bash -c "xdotool getactivewindow key Home"`\\
name : home+shift\\
command : `bash -c "xdotool key Shift+Home"`\\
name : end+shift\\
command : `bash -c "xdotool key Shift+End"`\\
를 하면 된다.
